Objeto

Um Objeto é uma instância de uma classe. Ou seja, quando você usa uma classe para criar algo real, aquilo é um objeto. O objeto é “algo concreto”: possui valores próprios para seus atributos e pode executar métodos.



Exemplo:



A classe Carro é o molde.
Um objeto seria:

cor = vermelho

modelo = Onix

ano = 2021
